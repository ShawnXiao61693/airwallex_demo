# 周报上传接口（给制作/上传 agent）

把人工编排好的周报 HTML 上传到平台，它会作为**草稿**进入后台「周报审核」，由 RevE 预览后点「通过发布」。

- 服务器：`http://13.214.205.219`
- 鉴权 token：在服务器 `~/airwallex/engine/.env` 的 `API_TOKEN`（或找 Shawn 要）

---

## 1. 上传周报 → 草稿

```
POST http://13.214.205.219/api/weekly
Header: X-Api-Token: <API_TOKEN>
Content-Type: multipart/form-data
```

| 字段 | 必填 | 说明 |
|---|---|---|
| `period` | 是 | 当周**周一**日期 `YYYY-MM-DD`，作期数标识 + 发布日 |
| `title` | 否 | 周报标题（后台列表显示用）|
| `file` | 是 | 周报 HTML 文件 |

**当前可用 period（2026 年 6–7 月，每周一）**：
`2026-06-01, 06-08, 06-15, 06-22, 06-29, 07-06, 07-13, 07-20, 07-27`

**curl 示例**：
```bash
curl -X POST http://13.214.205.219/api/weekly \
  -H "X-Api-Token: <API_TOKEN>" \
  -F "period=2026-06-22" \
  -F "title=第26周·费率战来了" \
  -F "file=@weekly_2026-06-22.html"
```

**python 示例**：
```python
import requests
requests.post("http://13.214.205.219/api/weekly",
  headers={"X-Api-Token": TOKEN},
  data={"period":"2026-06-22","title":"第26周·费率战来了"},
  files={"file": open("weekly.html","rb")})
```

**响应**：`{"ok":true,"period":"2026-06-22","status":"draft","url":"/airwallex/data/weekly/2026-06-22.html"}`

上传后：后台 http://13.214.205.219/airwallex/admin.html →「周报审核」对应周变为**草稿**，可预览、可点「通过发布」。

---

## 2. ⚠️ HTML 资产路径（重要约束）

上传的周报会被服务在 **`/airwallex/data/weekly/<period>.html`**。所以页面里引用平台样式必须用**绝对路径**，不能用相对路径：

```html
<!-- ✅ 对 -->
<link rel="stylesheet" href="/airwallex/assets/style.css">
<!-- ❌ 错（会 404，因为相对到了 data/weekly/assets/）-->
<link rel="stylesheet" href="assets/style.css">
```

或者把 CSS/JS **内联**进 HTML，做成完全自包含的单文件（更稳）。
（仓库里现成的 `web/weekly.html` 用的是相对路径，迁过来上传前需把资产路径改成绝对路径或内联。）

---

## 3. （可选）直接发布 / 查状态

- **发布**（一般由后台点按钮触发，也可调用，无需 token）：
  ```
  POST /api/publish   form: period=2026-06-22 & type=weekly
  ```
- **查所有周期 slots 及状态**：
  ```
  GET /api/slots?type=weekly
  → {"slots":[{"period":"2026-06-22","state":"empty|draft|published","title":"","url":...}, ...]}
  ```

---

## 4. 约束 / 注意
- 同一 `period` 重复上传 = 覆盖该周草稿（状态重置为 draft）。
- `period` 必须是 `YYYY-MM-DD`；落在 6–7 月周一之外的也能传，但不会在当前 slots 列表里显示。
- 上传走 token；发布暂不需要（demo 简化，生产应把后台放登录后）。
- 媒体（AI 播客/视频）目前为占位；对外前请复核文案与数字真伪。
