# skills —— 可复用 Agent Skill

本目录专门存放本项目的 **Agent Skill**：把"反复要做的、含人类判断的流程"固化成可复用的步骤，供制作/上传 agent 照着执行。每个 skill 一个子目录，内含 `SKILL.md`。

| Skill | 作用 |
|---|---|
| [上线报告](上线报告/SKILL.md) | 把做好的**周报 / 月报** HTML 通过 `/api/reports` 上传上线（草稿 → 人审 → 发布），含资产路径、媒体嵌入、验证、发布纪律 |

> 规划中（待数据库/流程就绪再写）：`制作周报`（从情报池选题→转化→出题→套模板，半自动产周报初稿）。
